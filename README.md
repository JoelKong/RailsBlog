# Rails Blog
A simple CRUD blog mainly to practice on ruby on rails

![rails](/assets/rails.PNG)

## Application Code
### Technology Stack
Front-End: ReactJS (TypeScript)  
Back-End: Ruby on Rails with SqlLite3
